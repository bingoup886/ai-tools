## 🔄 反馈流程

### 如果预览版本满意

1. 创建 Pull Request
   ```bash
   # 在 GitHub 上从 preview 创建 PR 到 main
   ```

2. 代码审查 (可选)

3. 合并到 main
   ```bash
   git checkout main
   git pull origin main
   git merge preview
   git push origin main
   ```

4. 生产部署
   ```
   Cloudflare 自动部署到生产环境
   ```

### 如果需要调整

1. 在 preview 分支继续修改
   ```bash
   git checkout preview
   # 进行修改
   git add .
   git commit -m "fix: 调整..."
   git push origin preview
   ```

2. Cloudflare 自动重新部署预览版本

3. 重新测试
